
**What is an operating system (OS)?**
It's a collection of fundamental software (kernels, device drivers, file managers,...) that manages a computer's hardware and applications while also acting as a medium between the user and the hardware. It's goal: provide a space where a user can execute programs conveniently and efficiently.

**What is the OS's role?**
- It's a resource allocator, managing resources such as CPU time, memory space, file-storage space, I/O devices, and decides how to allocate them to specific programs efficiently. It also functions as a control program, making sure of a smooth experience for the user.
- Provide the user a UI, allowing easy interactions

**Functions**
Process management: managing concurrently executing processes and threads, including scheduling them on the CPU and providing mechanisms for communication and synchronization.
Memory management: track the using memory, allocate/deallocate memory space, manage virtual memory
File-System management: abstracting physical storage into files, manage access rights and permissions
Device management: provide a uniform interface to various I/O devices through device drivers, enable the OS to communicate with hardware likes disks, keyboards and printers.

Examples:
- Windows (the most commonly used btw)
- Android, IOS
- Linux
